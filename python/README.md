# Python

## ライブラリ

[SPARQLWrapper](https://rdflib.github.io/sparqlwrapper/)

```
$ pip install -r requirement.txt
```

または

```
$ pip install sparqlwrapper
```

## 実行 (Python2)

```
$ python src/select1.py
```

## 実行 (Python3)

```
$ python src/select1_v3.py
```
