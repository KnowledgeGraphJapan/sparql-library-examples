# sparql-library-examples

SPARQLを各プログラム言語から使うためのサンプル集

## 現在サポートしている言語

* Java (java/)
* JavaScript (js/)
* JavaScript + AJAX (js-ajax/)
* Common Lisp (lisp/)
* PHP (php/)
* Python (python/)
* Ruby (ruby/)
